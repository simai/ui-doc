# Local variables

Local variables apply only within a specific plugin or component. They control parameters unique to that piece without
affecting the rest of the framework. Names follow: `--sf-{property}–{value}`.

**Examples:**

* `--sf-padding--small` — padding inside a plugin.
* `--sf-font-size--heading` — heading font size for a specific component.
