# Change background on hover

Use `.hover:bg-theme-{tone}` to change the background on hover.

```html
<button class="theme-dark bg-theme-3 hover:bg-theme-5 p-1 radius-1/3">
    Button with hover background (dark theme)
</button>
```

Themes make it easy to switch light/dark modes and reuse named colors and palettes for precise styling.
