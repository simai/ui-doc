# Using a theme

Use `.theme-light` for the light theme and `.theme-dark` for the dark theme.

```html
<div class="theme-light ...">
    <!-- Light theme content -->
</div>

<div class="theme-dark ...">
    <!-- Dark theme content -->
</div>
```
