# Named colors

Within a theme you can use these named colors:

- `.color-primary` — primary text color
- `.color-secondary` — secondary text color
- `.color-hint` — hint text color
- `.c-icon-active` — active icon color
- `.c-icon-inactive` — inactive icon color
- `.b-divider` — divider/border color

Classes adapt automatically to the light or dark palette.
