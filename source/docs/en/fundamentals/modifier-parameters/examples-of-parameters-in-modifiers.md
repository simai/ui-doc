# Parameter examples in modifiers

1. **Overflow:** `overflow-y-auto`, `overflow-x-hidden`.
2. **Positioning:** `inset-x-0`, `inset-y-0`.
3. **Gaps (flex/grid):** `gap-x-0`, `gap-y-2`.
4. **Scale:** `scale-x-0`, `scale-y-3`.
5. **Translate:** `translate-x-0`, `translate-y-1`.
6. **Skew:** `skew-x-0`, `skew-y-4`.

These parameters keep modifier usage flexible and precise.
