# Axis designations

Use these axis shortcuts:

1. `x` — horizontal axis
2. `y` — vertical axis
3. `main` — main axis (for flex/grid)
4. `cross` — cross axis (for flex/grid)

Add the axis after the property name with a dash. Use `x`/`y` for most elements; `main`/`cross` are for flex and grid.
