# Modifier parameters

The parameter clarifies where a modifier applies. It is added right after the property name and separated by a dash (`-`).
Parameters narrow the scope of a modifier and help target UI areas precisely. SIMAI uses three parameter types:

1. **Axis**
2. **Side**
3. **Corner**
