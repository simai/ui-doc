# Corner parameter

The corner parameter combines two sides (horizontal and vertical):

1. `left-bottom`
2. `left-top`
3. `right-bottom`
4. `right-top`

Add the corner after the property name with a dash. Examples:

1. **Positioning:** `left-top-auto`, `right-bottom-0`.
2. **Border radius:** `radius-left-bottom-1/3`, `radius-right-top-0`.

This helps control corner positioning and rounding precisely.
