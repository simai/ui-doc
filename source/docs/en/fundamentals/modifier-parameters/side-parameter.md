# Side parameter

Use the side parameter to target a specific side:

1. `left`
2. `right`
3. `top`
4. `bottom`

Add the side after the property name with a dash. Examples:

1. **Positioning:** `left-1/2`, `right-0`, `top-auto`, `bottom-2`.
2. **Border radius:** `radius-left-1/3`, `radius-top-0`.
3. **Margin:** `m-top-0`, `m-left-1`.
4. **Padding:** `p-bottom-1`, `p-right-1`.

**Note:** Side parameters resemble alignment values. A parameter is always followed by a value.

**Parameter:** `radius-left-1/3`, `top-1/2`

**Value:** `text-left`, `bg-top`
