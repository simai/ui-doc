# Property abbreviations

While modifiers favor full property names, some frequently used ones have shortcuts:

1. `bg` — background (`background`).
2. `h` — height (`height`).
3. `gr` — gradient (`gradient`).
4. `m` — margin (`margin`).
5. `p` — padding (`padding`).
6. `w` — width (`width`).
7. `z` — z-index.

These abbreviations make modifier usage more efficient and convenient, keeping interfaces flexible and adaptable.
