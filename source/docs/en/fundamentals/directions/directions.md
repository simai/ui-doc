# Directions

Axis-related values use these markers:

* `x` — horizontal.
* `y` — vertical.

Examples:

* `bg-repeat-x` — repeat background horizontally.
* `resize-y` — resize element vertically only.
* `snap-x` — control scroll snapping horizontally.
