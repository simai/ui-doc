# Relationship between letters and multipliers

**Mapping between letters and multipliers:**

* A: 0
* B: 1
* C: 2
* D: 4
* E: 8
* F: 16
* G: 32
* H: 64
* I: 128

Starting from B, each subsequent letter doubles the base multiplier.
