````markdown
# Using themes

To use the light theme, apply the `.theme-light` class; for the dark theme use `.theme-dark`.

```html
<div class="theme-light ...">
    <!-- Содержимое со светлой темой -->
</div>

<div class="theme-dark ...">
    <!-- Содержимое с темной темой -->
</div>
```

````
