# Theme color palette

To use the theme color palette, specify the class: `.{prefix}-theme-{intensity}`

- Prefix – required parameter:

    - `.color-...` for text color
    - `.bg-...` for background color
    - `.border-...` for border color
- Intensity – optional parameter (from 1 to 9).
