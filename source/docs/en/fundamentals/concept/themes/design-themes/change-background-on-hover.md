# Change background on hover

With the `.hover:bg-theme-{intensity}` class you can set a background change when the cursor hovers over an element.

```html
<button class="theme-dark bg-theme-3 hover:bg-theme-5 p-1 radius-1/3">
    Кнопка с изменением фона при наведении (темная тема)
</button>
```

Thus, design themes make it easy to switch between light and dark modes and use named colors and palettes for fine
tuning of the interface appearance.
