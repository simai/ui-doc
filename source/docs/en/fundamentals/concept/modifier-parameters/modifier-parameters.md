# Modifier parameters

A modifier parameter defines the area on which a given modifier will act. It is added immediately after the property name and is separated by a hyphen (`-`). Parameters clarify the scope of a modifier and help you precisely control interface elements. SIMAI Framework uses the following parameter types:

1. **Axis**
2. **Side**
3. **Angle**
