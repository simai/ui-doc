# Examples of parameters in modifiers

1. **Overflow**: `overflow-y-auto`, `overflow-x-hidden`.
2. **Positioning**: `inset-x-0`, `inset-y-0`.
3. **Gaps (flex and grid)**: `gap-x-0`, `gap-y-2`.
4. **Scaling**: `scale-x-0`, `scale-y-3`.
5. **Translation**: `translate-x-0`, `translate-y-1`.
6. **Skew**: `skew-x-0`, `skew-y-4`.

This approach provides flexibility and precision when tuning interface elements.
