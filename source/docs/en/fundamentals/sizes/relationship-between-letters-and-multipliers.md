# Relationship Between Letters and Multipliers

**Mapping letters to multipliers:**

* A: 0
* B: 1
* C: 2
* D: 4
* E: 8
* F: 16
* G: 32
* H: 64
* I: 128

Starting with B, each next letter doubles the base multiplier.
