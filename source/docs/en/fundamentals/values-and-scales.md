# Values

Modifiers in SIMAI Framework can take various types of values that make it possible to flexibly configure interface
elements. The value types include:

* Absolute dimensions
* Relative sizes
* Proportional values
* Binary values
* Percents
* Degrees
* Colors
* Directions
* Alignment
* Named values
