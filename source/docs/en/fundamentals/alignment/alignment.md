# Alignment

Use these terms for positioning and aligning elements:

* `left` — left.
* `right` — right.
* `center` — centered.
* `justify` — justified.
* `top` — top.
* `bottom` — bottom.
* `middle` — vertical center.
* `start` — start (respecting writing direction).
* `end` — end (respecting writing direction).

Corner alignment combines horizontal and vertical directions:

* `left-top` — top-left corner.
* `left-bottom` — bottom-left corner.
* `right-bottom` — bottom-right corner.
* `right-top` — top-right corner.
