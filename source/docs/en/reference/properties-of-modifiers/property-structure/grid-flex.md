# Grid and flex (grid-flex)

* Item order (order).
* Gaps between items (gap).
* Align content along the main axis (justify-content).
* Align all items along the main axis (justify-items).
* Align a single item along the main axis (justify-self).
* Align content along the cross axis (align-content).
* Align all items along the cross axis (align-items).
* Align a single item along the cross axis (align-self).
* Place content along both axes (place-content).
* Place all items along both axes (place-items).
* Place a single item along both axes (place-self).
* Alignment along both axes (flex-align).
