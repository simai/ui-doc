# Main roles

* **Surface** — background role for large areas (site background or its parts). It has a low accent and is based on
  neutral colors.
* **Primary** — the main accent color that highlights the most important elements.
* **Secondary** — a secondary accent color that is more muted. Used for frequently repeated interface elements.
* **Tertiary** — a color antagonist to primary that highlights elements next to primary, giving the composition visual
  balance.
* **Error** — color for indicating errors or serious problems.
* **Warning** — color for warnings.
* **Success** — color indicating a successful state.
