# Link

The **Link** role is intended for styling links in text, providing their visual highlighting and clear states on
interaction (hover, active, visited).

![][image26]

![][image27]

Variables for the Link role:

| Variable           | Value (light)    | Value (dark)     |
|:-------------------|:-----------------|:-----------------|
| `--sf-link`        | `--sf-primary-40`  | `--sf-primary-80`  |
| `--sf-link-hover`  | `--sf-primary-35`  | `--sf-primary-85`  |
| `--sf-link-active` | `--sf-primary-30`  | `--sf-primary-90`  |
| `--sf-link-visited`| `--sf-tertiary-40` | `--sf-tertiary-80` |
{.table}
