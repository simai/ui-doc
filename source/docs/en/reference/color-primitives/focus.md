# Focus

The Focus role is used to visualize focus on an element. In the current version of the framework, focus is implemented
as an outline. Additional variables for shadow size allow you to configure the display parameters when focused.

![][image28]

Variables for the Focus role:

| Variable           | Value (light)              | Value (dark)               |
|:-------------------|:---------------------------|:---------------------------|
| `--sf-focus--color`| `--sf-primary-50--alfa-24` | `--sf-primary-90--alfa-24` |
{.table}
