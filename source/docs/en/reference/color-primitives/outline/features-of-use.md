# Usage features

For the **Outline Variant**, semi-transparent colors are used, which makes it possible to place it over various
backgrounds with different contrast levels. Thus, the outline remains visible without increasing visual noise.

![][image24]

![][image25]

The following variables are used for the Outline role:

| Variable              | Value (light)              | Value (dark)               |
|:----------------------|:---------------------------|:---------------------------|
| `--sf-outline`        | `--sf-neutral-50`          | `--sf-neutral-60`          |
| `--sf-outline-variant`| `--sf-neutral-50--alfa-24` | `--sf-neutral-90--alfa-24` |
{.table}
