# Mark

The Mark role is used to highlight text, for example to emphasize important information or search results.

![][image29]

Variables for the Mark role:

| Variable   | Value (light)              | Value (dark)               |
|:-----------|:---------------------------|:---------------------------|
| `--sf-mark`| `--sf-warning-50--alfa-24` | `--sf-warning-90--alfa-24` |
