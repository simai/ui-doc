# Справочник компонентов

Здесь публикуются точные страницы по отдельным компонентам. Каждая запись
описывает один подтверждённый runtime-контракт, а не просто похожую разметку из
другого проекта.

## Что будет в каждой записи

- назначение и граница компонента;
- идентификатор правила и способ распознавания разметки;
- необходимые ассеты и зависимости;
- минимальный проверенный пример;
- локальное поведение и доступность;
- связанные Smart Components, если они есть;
- source paths и release lock, на которых подтверждена запись.

Перед публикацией отдельной страницы проверьте её на утверждённом lock в
браузере. Если source evidence описывает только ассет или внутреннюю
зависимость, это не основание для самостоятельной пользовательской страницы.

Вернитесь к [каталогу компонентов](/ru/components/catalog/), чтобы выбрать
элемент, или к [примерам](/ru/components/examples/), чтобы подготовить
минимальный сценарий.
