# Оформление таблицы

!rtags[table]


## Таблица классов

| Класс | Значение |
|:--|:--|
| `.table` | Базовое оформление таблицы |
| `.table-sm` | Компактные отступы ячеек |
{.table}

## Пример

```html
<table class="table">...</table>
<table class="table table-sm">...</table>
```
