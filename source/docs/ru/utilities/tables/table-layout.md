# Макет таблицы

!rtags[table-layout]


## Таблица классов

| Класс | Значение |
|:--|:--|
| `.table-auto` | `table-layout: auto;` |
| `.table-fixed` | `table-layout: fixed;` |
{.table}

## Пример

```html
<table class="table table-auto">...</table>
<table class="table table-fixed">...</table>
```
