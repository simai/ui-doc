# Границы таблицы

!rtags[table-border]


## Таблица классов

| Класс | Значение |
|:--|:--|
| `.table-border` | Добавляет границы таблицы и ячеек |
{.table}

## Пример

```html
<table class="table table-border">...</table>
```
