# Цвет текста

## Таблица классов

| Класс | Значение |
|:--|:--|
| `.color-on-surface` | Цвет текста по токену on-surface |
| `.color-primary` | Цвет текста по токену primary |
| `.color-warning` | Цвет текста по токену warning |
| `.color-success` | Цвет текста по токену success |
{.table}

## Пример

```html
<p class="color-on-surface">Base text</p>
<p class="color-primary">Primary text</p>
<p class="color-warning">Warning text</p>
```
