# Точки градиента (gradient-stops)

!rtags[background-gradient-stops]


Утилиты `from-*`, `via-*`, `to-*` задают цветовые точки градиента:
- `from-*` -> `--sf-gradient--color-1`
- `via-*` -> `--sf-gradient--color-2`
- `to-*` -> `--sf-gradient--color-3`
