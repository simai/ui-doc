# Прозрачность кольца (ring-opacity)

!rtags[ring-opacity hover focus]


Утилиты `ring-opacity-*` задают прозрачность цвета кольца.

## Классы

| Класс | Значение |
|:--|:--|
| `.ring-opacity-0 ... .ring-opacity-9` | `opacity: 0 ... 0.9` |
| `.ring-opacity-full` | `opacity: 1` |
{.table}
