# Быстрый старт

Быстрый старт начинается не с копирования случайной разметки, а с одной
проверяемой пары assets. Пока для baseline `5.4.0` не опубликован approved
lock, эта страница описывает порядок действий, но не выдаёт шаблон с
плейсхолдером за рабочий production-код.

1. Выполните [установку](/ru/start/installation/) по Core ref из lock.
2. Откройте approved example, связанный с тем же lock.
3. Скопируйте только разметку и классы, которые есть в release catalog.
4. Проверьте Console, Network и responsive-состояния в своём проекте.
5. Перейдите к [Основам](/ru/fundamentals/), чтобы понимать условия и
   ограничения модификаторов.

Playground удобен для исследования, но не заменяет pinned пример из release
catalog.
