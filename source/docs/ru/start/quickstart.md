# Быстрый старт

Короткий маршрут для первого знакомства:

1. Подключите CSS и JS так, как описано в [установке](installation.md).
2. Откройте любой пример из [Playground](playground.md) и скопируйте разметку.
3. Добавьте несколько базовых модификаторов (отступы, цвета, типографика) и проверьте адаптивность через контрольные точки.

Дальше переходите к разделу [Основы](../fundamentals/index.md) и изучайте сетки и брейкпоинты, чтобы собирать адаптивные интерфейсы.
