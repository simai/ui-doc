# Глоссарий

**SIMAI Framework** — набор фронтенд-утилит, компонентов и Smart Components
для создания интерфейсов. Конкретный состав поставки и её совместимость
определяются approved release lock.

**Модификаторы (утилиты)** — CSS-классы, которые задают свойства HTML-элемента:
например, отступ, цвет, размер или режим раскладки.

**Адаптивные модификаторы** — утилиты, значение которых зависит от контрольной
точки. Их фактический синтаксис и список значений сверяются с release catalog.

**CSS (Cascading Style Sheets)** — язык описания внешнего вида HTML-документа.

**UI (User Interface)** — пользовательский интерфейс: визуальная часть
веб-приложения и механизмы взаимодействия с ней.

**HTML (HyperText Markup Language)** — язык разметки структуры веб-страницы.

**JavaScript (JS)** — язык программирования для поведения интерфейса в
браузере.

**API (Application Programming Interface)** — публичный контракт: методы,
события, атрибуты или форматы, на которые может опираться интеграция. В этой
документации API считается текущим только если он включён в release catalog.

**Flexbox** — CSS-модель раскладки для выравнивания и распределения элементов
в одной строке или колонке.

**Grid Layout** — CSS-модель раскладки по строкам и колонкам.

**Mobile First** — подход, в котором базовые стили рассчитаны на небольшой
экран, а затем уточняются для более широких экранов.

**Компонент** — повторно используемый блок интерфейса с определённым
контрактом подключения и поведения.

**Smart Components** — слой custom elements и связанных runtime-артефактов.
Его зависимости, атрибуты и assets должны быть подтверждены для конкретного
release lock.

**Фронтенд** — клиентская часть приложения: разметка, CSS, JavaScript и
логика взаимодействия в браузере.

**Примитивы (Primitives)** — базовые значения дизайн-системы, например палитра
или шкала размеров.

**Токены (Tokens)** — именованные значения, построенные на примитивах и
используемые для внешнего вида или поведения интерфейса.

**Цветовая схема (Color Scheme)** — набор токенов, описывающих используемые
цвета. Их состав определяется дизайн-системой и release catalog.
