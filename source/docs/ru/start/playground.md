# Playground

Быстрые способы пощупать модификаторы без развертывания проекта:

- Встроенные фреймы из [play.simai.io](https://play.simai.io) — копируйте примеры и меняйте модификаторы прямо в браузере.
- Включите `sf-playground` на любой странице и вставьте демо-код из раздела [Utilities](../utilities/index.md).
- Для локальных экспериментов держите рядом `webpack.mix.js`/`vite`-конфиг и обновляйте сборку после правок классов.

Если что-то не работает, проверьте сеть на загрузку ассетов и сравните с [настройками проекта](project-setup.md).
