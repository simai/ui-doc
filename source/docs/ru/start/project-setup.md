# Настройка проекта

Рекомендуемая последовательность для рабочего окружения:

1. Синхронизируйте зависимости фреймворка и утилит (Composer/NPM) и проверьте версии сборщика.
2. Держите базовую структуру `resources`/`public` с путями к ассетам, которые ожидает загрузчик.
3. Включите линтеры и форматирование для Blade/Markdown, чтобы примеры из документации не расходились со стилем проекта.
4. Настройте дев-сервер с горячей перезагрузкой, чтобы быстро проверять модификаторы и утилиты.

После конфигурации переходите к [Fundamentals](../fundamentals/index.md) и настройке [Loader](loader/index.md) под своё окружение.
