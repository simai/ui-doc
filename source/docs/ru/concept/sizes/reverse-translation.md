# Обратный перевод (из пикселей к буквенно-цифровому)

1. **Определить зону**:  
   Допустим, есть 56px.  
   Смотрим пороги: 56 \>40 и \<80, значит зона D.

2. **Вычесть старт**:  
   Для D это 40, получаем 56–40=16.

3. **Разделить на множитель**:  
   Множитель для D=4.  
   16 ÷4=4 → значит `d4`.
