# Изменение фона при наведении

С помощью класса `.hover:bg-theme-{насыщенность}` можно задавать изменение фона при наведении курсора.

```html
<button class="theme-dark bg-theme-3 hover:bg-theme-5 p-1 radius-1/3">
    Кнопка с изменением фона при наведении (темная тема)
</button>
```

Таким образом, темы оформления позволяют легко переключаться между светлым и темным режимами, а также использовать
именные цвета и палитры для точной настройки внешнего вида интерфейса.
