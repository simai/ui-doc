# Использование темы

Для использования светлой темы примените класс `.theme-light`, для темной — `.theme-dark`.

```html
<div class="theme-light ...">
    <!-- Содержимое со светлой темой -->
</div>

<div class="theme-dark ...">
    <!-- Содержимое с темной темой -->
</div>
```
