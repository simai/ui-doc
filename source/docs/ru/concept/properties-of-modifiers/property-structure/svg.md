# SVG (svg)

* Размер изображения (svg-size).
* Цвет заливки (fill).
* Тип заливки (fill-rule).
* Цвет обводки (stroke-color).
* Толщина обводки (stroke-width).
* Углы обводки (stroke-linejoin).
* Концы обводки (stroke-linecap).
