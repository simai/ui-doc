# Описание

SFLoader — это клиент-серверная система динамической загрузки компонентов, CSS/JS-файлов и шаблонов.  
Она включает в себя фронтенд-библиотеку SFLoaderPlugin и серверную архитектуру на PHP с поддержкой кэширования, шаблонов
и зависимостей.
